Enunciado

Professor Thiago quer ajudar seus alunos a entender o conceito de múltiplos de um número. Ele deseja uma função que possa gerar os primeiros n múltiplos de um número x. Escreva uma função que receba dois inteiros, x (o número base) e n (o número de múltiplos a gerar), e use um loop while para criar uma lista com os primeiros n múltiplos de x.

Entrada: Dois números inteiros, n e x.

Saída: Uma lista contendo os primeiros n múltiplos de x.
